Word Office Add-in

OpenRedline

在 Word 里比较多个 AI 的文本修订结果。读取选中文本，使用可自定义 Prompt 和模型配置生成多个版本，人工比较后再替换回文档。

分发路线

个人或小范围测试

使用本地 manifest，让每位测试者在自己的电脑上运行服务并填入 API 配置。

团队内部使用

把前端和后端部署到同一个 HTTPS 域名，再用生产版 manifest 指向线上地址。

公开发布

补齐隐私政策、支持页面和权限说明后，再考虑 Microsoft AppSource 审核。
